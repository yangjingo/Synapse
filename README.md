## Overview
Synapse is a Claude Skill that transforms technical notes into high-end engineering blogs and "Why.J Theater" slides. It bridges the gap between deep logic and world-class presentation.

## Installation
```bash
claude plugin install https://github.com/yangjingo/Synapse
```

## Core Capabilities
- **Research Blog**: Academic rigor with OpenAI/Anthropic styles.
- **Why.J Engineering Theater**: O(1) complexity HTML slides.
- **Social Distro**: Hardcore tech content for X, Red, and Zhihu.

## Directory Structure
- `SKILL.md`: Main instruction set for AI.
- `references/`: Visual assets, style guides, and resource library.
- `assets/templates/`: Markdown templates for blogs.
- `scripts/`: Technical utilities.
